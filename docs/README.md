Archive documentation when:
- Implementation phase is complete and documented elsewhere
- Strategy/feature is deprecated or replaced
- Content is outdated by more than 2 months

### Naming Convention
Archive folders use format: `YYYY-MM-description/`
- Example: `2025-09-paper-trading/`
